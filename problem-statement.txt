Assessment Title: Ethara Seat Allocation & Project Mapping System

Objective

Build a full-stack application that manages seat allocation and
project mapping for approximately 5,000 employees. The solution should
enable employees, HR, Admin, and Project teams to efficiently manage
and search employee seating, project assignments, seat availability,
utilization metrics, and new joiner allocations.

Key Requirements
Employee Management
Project Mapping
Seat Allocation & Release
New Joiner Seat Allocation
Search & Filter Functionality
Dashboard & Analytics
AI Assistant / Natural Language Query Interface
REST APIs
Seed Data Generation
Recommended Tech Stack
Frontend: React.js / Next.js + Tailwind CSS
Backend: FastAPI
Database: PostgreSQL (preferred)
Deployment: Railway, Render, Vercel, Netlify, or Fly.io

You are free to use AI tools such as ChatGPT, Claude, Cursor, GitHub
Copilot, Gemini, Replit AI, or similar tools during development.
However, all AI usage must be documented in an AI_PROMPTS.md file,
including prompts used, outputs generated, manual fixes applied, and
validation methods.

Submission Requirements

Please share the following upon completion:

GitHub Repository URL
Live Frontend URL
Live Backend URL
README.md
AI_PROMPTS.md
Database Schema
Seed Data
API Documentation / Swagger URL
Screenshots
Deployment Notes
Debugging Notes
